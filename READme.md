# Sentiment Analysis AI 

Mon premier projet IA qui analyse si un commentaire est positif ou négatif.

## Comment tester
1. Cloner le dépôt
2. Installer Python et les librairies : `pip install scikit-learn pandas`
3. Lancer : `python interface.py`

## Fonctionnalités
- Prédit positif / négatif
- Interface graphique Tkinter
- Facile à utiliser pour tester vos phrases
